Objectif : apprendre à utiliser les feuilles de style : la syntaxe de base, les styles contextuels, les classes de style, les divisions.

A retenir: Les feuilles de style servent à séparer le contenu (inscrit dans le fichier (X)HTML) et la forme, qui est contenue dans le fichier de style. Elles sont particulièrement utiles pour homogénéïser le style donné à plusieurs pages HTML. Elles facilitent également la maintenance de ces pages (par exemple, celles qui forment un site Web) car il suffit de la modifier pour modifier instantanément le style de toutes les pages liées à la feuille de style. Les divisions permettent de définir des conteneurs flottants, dont l'agencement souple permet d'améliorer le rendu des sites Web.
